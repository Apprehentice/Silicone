Class Root

The root level element of all Silicone menu structures.

The Root element is the foundation for any Silicone menu hierarchy as it handles all of Silicone's callbacks and fires them for its children.

Not all available methods are listed here. For more information, see Base.

Internal

Root:initialize (spec) Initializes a Root element
Root:_add_name (element) Adds a named element to the Root's name index
Root:_restack_focus () Reorders the focus stack so that it can be indexed.

Querying

Root:find (key) Finds an element with the given name.

Getters/Setters

Root:getAbsoluteWidth () Returns a Root element's width (the screen width)
Root:getAbsoluteHeight () Returns a Root element's height (the screen height)
Root:getAbsoluteX () Returns a Root element's X coordinate (usually 0)
Root:getAbsoluteY () Returns a Root element's Y coordinate (usually 0)

Focus Stack

Root:pushFocus (element, norestack) Pushes an element onto the focus stack and puts it in focus
Root:popFocus (norestack) Pops the first element from the focus stack
Root:peekFocus (norestack) Returns the first element on the focus stack
Root:focus () Sets focus to the first available element in the focus stack

Events

Root:update (dt) Updates all of a Root's children and resizes it to fit the screen.
Root:doLeft () Calls onLeft for the currently focused element
Root:doRight () Calls onRight for the currently focused element
Root:doUp () Calls onUp for the currently focused element
Root:doDown () Calls onDown for the currently focused element
Root:doConfirm () Calls onConfirm for the currently focused element
Root:doCancel () Calls onCancel for the currently focused element

Callbacks

Root:onFocus () Does nothing
Root:onLeft () Does nothing
Root:onRight () Does nothing
Root:onUp () Does nothing
Root:onDown () Does nothing
Root:onConfirm () Does nothing
Root:onCancel () Does nothing
